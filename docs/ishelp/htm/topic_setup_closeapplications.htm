[Setup]: CloseApplications


Допустимые значения:
force, yes или no

Значение по умолчанию:
yes

Описание:

При значении yes или force и работе инсталлятора в обычном режиме, программа установки приостановит свою работу на странице "Подготовка к установке", если обнаружит, что файлы секции [Files] или [InstallDelete], нуждающиеся в замене/обновлении, используются другими приложениями. При этом будет показан список таких приложений, а также выдан запрос о необходимости автоматического закрытия приложений и возможности их перезапуска после завершения установки.

При значении yes или force и работе инсталлятора в тихом режиме, программа установки всегда будет закрывать и перезапускать такие приложения, если через командную строку не будут переданы параметры отмены.

При значении force, программа установки принудительно закроет приложения, если через командную строку не будут переданы параметры отмены. Используйте это значение с осторожностью, т.к. пользователь может потерять несохранённые данные своей работы.

Примечание: для закрытия и перезапуска приложений программа установки использует функции API Windows механизма Restart Manager.
