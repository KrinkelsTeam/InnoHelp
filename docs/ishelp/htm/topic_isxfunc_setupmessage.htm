Pascal Scripting: SetupMessage

Прототип

function SetupMessage(const ID: TSetupMessageID): String;

Описание

Возвращает строку сообщения программы установки, соответствующее указанной константе ID.

Пример

var
  S: String;
begin
  S := SetupMessage(msgButtonNext);
  // Теперь переменная S содержит строку '&Next >'
end;
