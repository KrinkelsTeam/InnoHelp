[Setup]: UsePreviousPrivileges


Допустимые значения:
yes или no

Значение по умолчанию:
yes

Описание:

При значении yes программа установки во время своего запуска выполнит в реестре поиск информации об установке того же приложения в одном из двух режимов и, если таковая будет найдена, использует предыдущий режим установки без запроса к пользователю.

Если PrivilegesRequiredOverridesAllowed не имеет значение dialog, эта директива не учитывается.

Директива UsePreviousPrivileges должна иметь значение no, если AppId содержит константы, и PrivilegesRequiredOverridesAllowed равна dialog.
