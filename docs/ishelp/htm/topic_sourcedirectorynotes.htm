Исходный каталог


По умолчанию компилятор выполняет поиск файлов, упомянутых в секциях [Files] и [Setup], в том же каталоге, в котором находится файл сценария, если только в параметрах не используется полный путь. Чтобы использовать другой каталог с исходными файлами, добавьте в сценарий директиву SourceDir в секции [Setup].
