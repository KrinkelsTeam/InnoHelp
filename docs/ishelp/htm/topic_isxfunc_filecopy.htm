Pascal Scripting: FileCopy

Прототип

function FileCopy(const ExistingFile, NewFile: String; const FailIfExists: Boolean): Boolean;

Описание

Выполняет копирование существующего файла ExistingFile в новый файл NewFile, сохранив отметку времени и атрибуты файла.

Если параметр FailIfExists установлен в True и новый файл NewFile уже существует, то функция не выполнится, в противном случае файл будет перезаписан.

Возвращает значение True, если копирование было выполнено успешно, иначе False.
