Pascal Scripting: ExtractFilePath


Прототип:

function ExtractFilePath(const FileName: string): String;

Описание:

Извлекает диск и каталог для указанного имени файла. Итоговая строка будет содержать все символы левой части FileName, включая двоеточие или обратную косую черту, которые отделяют имя пути от имени файла с расширением. Если заданное имя не содержит определение пути (диск и каталог), то результатом будет пустая строка.
