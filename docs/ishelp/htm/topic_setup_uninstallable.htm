[Setup]: Uninstallable

Допустимые значения:
yes или no, или логическое выражение в сценарии

Значение по умолчанию:
yes

Описание

Указывает компилятору о необходимости включения в инсталлятор программы автоматического удаления приложения (деинсталлятор). Если указано значение yes или логическое выражение в сценарии вернёт значение True, деинсталлятор будет создан. Иначе автоматическое удаление будет недоступно, и конечному пользователю придётся вручную удалять файлы, относящиеся к вашему приложению.

Использование логического выражения в качестве значения может быть полезным, если вы хотите предложить пользователю опцию "портативной установки".

Пример:
[Setup]
Uninstallable=not IsTaskSelected('portablemode')

[Tasks]
Name: portablemode; Description: "Portable Mode"
