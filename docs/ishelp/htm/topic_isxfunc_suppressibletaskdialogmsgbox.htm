Pascal Scripting: SuppressibleTaskDialogMsgBox


Прототип:

function SuppressibleTaskDialogMsgBox(const Instruction, Text: String; const Typ: TMsgBoxType; const Buttons: Cardinal; const ButtonLabels: TArrayOfString; const ShieldButton: Integer; const Default: Integer): Integer;

Описание:

Выводит на экран подавляемое окно сообщения, используя диалог задач. Если окна сообщений подавляются (смотрите "Параметры командной строки программы установки"), то функция возвращает значение, заданное параметром Default. В противном случае поведение функции SuppressibleTaskDialogMsgBox аналогично функции TaskDialogMsgBox.
