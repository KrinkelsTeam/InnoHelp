Pascal Scripting: PackVersionComponents

Прототип

function PackVersionComponents(const Major, Minor, Revision, Build: Word): Int64;

Описание

Выполняет упаковку указанных компонентов версии в одно значение.

Для сравнения упакованных версий всегда используйте функцию ComparePackedVersion или SamePackedVersion.
