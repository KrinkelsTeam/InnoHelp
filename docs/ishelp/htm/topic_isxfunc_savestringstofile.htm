Pascal Scripting: SaveStringsToFile

Прототип

function SaveStringsToFile(const FileName: String; const S: TArrayOfString; const Append: Boolean): Boolean;

Описание

Сохраняет указанный массив строк TArrayOfString в заданном файле FileName в кодировке ASCII. Если параметр Append имеет значение True и заданный файл уже существует, то строки будут добавлены к файлу, вместо его полной перезаписи. В случае успеха функция возвращает значение True, иначе — False.
