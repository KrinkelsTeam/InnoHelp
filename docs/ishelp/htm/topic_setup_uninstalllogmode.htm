[Setup]: UninstallLogMode


Допустимые значения:
append, new или overwrite

Значение по умолчанию:
append

Описание:

Значение append указывает программе установки по возможности дополнить существующий журнал деинсталляции.

Значение new указывает программе установки всегда создавать новый журнал деинсталляции. Такое поведение инсталлятора было в ранних версиях (до 1.3) Inno Setup.

Значение overwrite указывает программе установки перезаписать любые, существующие журналы деинсталляции того же приложения вместо их дополнения (крайне не рекомендуется). Те же правила добавления к существующим журналам применяются к перезаписи существующих журналов.

Пример:
UninstallLogMode=append
