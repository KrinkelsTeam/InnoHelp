[Setup]: MinVersion


Формат:
major.minor

Значение по умолчанию:
6.1sp1 (Windows 7 с SP1 или Windows Server 2008 R2 с SP1)

Описание:

Эта директива позволяет указать минимальную версию Windows, в которой программное обеспечение будет работать. Также могут быть указаны номера сборок и/или пакеты обновления.

Если система пользователя не соответствует минимальным требованиям к версии, программа установки выдаст сообщение об ошибке и завершит работу.
