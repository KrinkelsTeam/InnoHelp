Программирование: GetSHA1OfUnicodeString

Прототип

function GetSHA1OfUnicodeString(const S: String): String;

Описание

Возвращает строковое значение контрольной суммы SHA-1 для заданной строки в кодировке Unicode.

Пример

var
  SHA1: String;
begin
  SHA1 := GetSHA1OfUnicodeString('Test');
  // SHA1 = '9ab696a37604d665dc97134dbee44cfe70451b1a'
end;
