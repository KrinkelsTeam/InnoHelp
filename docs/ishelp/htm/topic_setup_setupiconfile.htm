[Setup]: SetupIconFile


Описание:

Эта директива позволяет указать другой значок, который будет использоваться программой установки/удаления. При сборке инсталлятора файл значка должен находиться в исходном каталоге сценария, в противном случае необходимо указать полный путь к файлу или путь с приставкой "compiler:", тогда компилятор будет искать его в собственном каталоге.

Пример:
SetupIconFile=MyProgSetup.ico
