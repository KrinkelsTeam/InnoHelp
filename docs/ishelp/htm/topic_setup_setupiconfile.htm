[Setup]: SetupIconFile


Значение по умолчанию:
(пустое)

Описание:

Эта директива позволяет указать другой значок, который будет использоваться программой установки/удаления. При сборке инсталлятора файл значка должен находиться в исходном каталоге сценария, в противном случае необходимо указать полный путь к файлу или путь с приставкой "compiler:", тогда компилятор будет искать его в собственном каталоге.

При создании собственного или использовании другого значка рекомендуется включать в него как минимум следующие размеры изображений: 16x16, 32x32, 48x48, 64x64 и 256x256.

Если эта директива не указана или имеет пустое значение, то будет использоваться встроенный значок, поддерживающий перечисленные выше размеры.

Чтобы вновь использовать старый значок программы установки, укажите директиву со значением compiler:SetupClassicIcon.ico.

Пример:
SetupIconFile=MyProgSetup.ico
