[Setup]: UserInfoPage


Допустимые значения:
yes или no

Значение по умолчанию:
no

Описание:

При значении yes программа установки отобразит страницу "Информация о пользователе", на которой нужно указать имя пользователя, организацию и, возможно, серийный номер. Введённые пользователем данные будут присвоены константам {userinfoname}, {userinfoorg} и {userinfoserial}. Если инсталляция выполняется в режиме не административной установки, эти константы можно применить в записях секции [Registry] или [INI], чтобы сохранить их значения для дальнейшего использования.

Чтобы поле серийного номера стало видимым, должна быть объявлена функция для события CheckSerial.

Директивы DefaultUserInfoName, DefaultUserInfoOrg и DefaultUserInfoSerial определяют для имени пользователя, организации и серийного номера значения по умолчанию. Если директива UsePreviousUserInfo имеет значение yes и программа установки найдет информацию об установке того же приложения, она будет использовать имя пользователя, организацию и серийный номер из предыдущей установки.

При тихой установке для имени пользователя, организации и серийного номера будут приняты значения по умолчанию. Программа установки не будет проверять, является ли имя пользователя пустым (так как у пользователя не будет возможности его исправить), однако серийный номер всё равно будет проверен.
