[Setup]: Output


Допустимые значения:
yes или no

Значение по умолчанию:
yes

Описание:

При значении no компилятор только проверит сценарий на наличие ошибок и пропустит создание файлов установки. Обратите внимание, что выходной каталог всё равно будет очищен, а файл манифеста удалён, пока для директив OutputDir и OutputManifestFile не будут указаны пустые строки.
