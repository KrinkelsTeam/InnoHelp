Pascal Scripting: GetOpenFileNameMulti


Прототип:

function GetOpenFileNameMulti(const Prompt: String; var FileNameList: TStrings; const InitialDirectory, Filter, DefaultExtension: String): Boolean;

Описание:

Выполняет вызов диалогового окна, в котором пользователь может выбрать один или несколько существующих файлов. Возвращает значение True, если пользователь подтвердил выбор файлов, иначе False. Имена выбранных файлов передаются в параметре FileNameList.

Примечание:

К примеру, фильтр файлов: 'Text files (*.txt)|*.txt|All files (*.*)|*.*'

Пример:
var
  FileNameList: TStrings;
begin
  // Создаём список
  FileNameList := TStringList.Create;
  try
    if GetOpenFileNameMulti('', FileNameList, '',
       'Текстовые файлы (*.txt)|*.txt|Все файлы|*.*', 'txt') then
    begin
      // Успех: пользователь нажал OK
      // Переменная FileName содержит выбранные имена файлов
    end;
  finally
    FileNameList.Free;
  end;
end;
