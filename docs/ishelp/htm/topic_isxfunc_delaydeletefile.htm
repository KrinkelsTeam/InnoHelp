Pascal Scripting: DelayDeleteFile

Прототип

procedure DelayDeleteFile(const Filename: String; const Tries: Integer);

Описание

Выполняет попытку удаления указанного файла Filename. Если файл используется, процедура повторяется до указанного значением Tries раз. Задержка между попытками удаления составляет 250 мс.
