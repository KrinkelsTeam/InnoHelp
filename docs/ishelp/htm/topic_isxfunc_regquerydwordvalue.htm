Pascal Scripting: RegQueryDWordValue


Прототип:

function RegQueryDWordValue(const RootKey: Integer; const SubKeyName, ValueName: String; var ResultDWord: Cardinal): Boolean;

Описание:

Запрашивает значение типа REG_DWORD в указанном разделе реестра и возвращает данные в параметре ResultDWord. В случае успеха возвращает значение True. Если функция возвращает False, то данные ResultDWord остаются без изменений.

Пример:
var
  HistoryBufferSize: Cardinal;
begin
  if RegQueryDWordValue(HKEY_CURRENT_USER, 'Console',
     'HistoryBufferSize', HistoryBufferSize) then
  begin
    // Успешно получили значение
    MsgBox('Размер буфера протокола консоли: ' + IntToStr(HistoryBufferSize),
      mbInformation, MB_OK);
  end;
end;
