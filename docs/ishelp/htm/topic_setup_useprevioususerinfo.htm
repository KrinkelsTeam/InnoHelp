[Setup]: UsePreviousUserInfo


Допустимые значения:
yes или no

Значение по умолчанию:
yes

Описание:

При значении yes программа установки во время своего запуска выполнит в реестре поиск информации об установке того же приложения и, если таковая будет найдена, использует имя пользователя, название организации и серийный номер, которые пользователь указал на странице "Информация о пользователе" во время предыдущей установки, в качестве настроек установки по умолчанию.

Обратите внимание, что программа установки не может повторно использовать настройки предыдущей установки, у которой директива Uninstallable имела значение no, поскольку искомые записи реестра, когда Uninstallable=no, не создаются.
