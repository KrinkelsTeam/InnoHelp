[Setup]: SignToolMinimumTimeBetween


Значение по умолчанию:
0

Описание:

Если не указано значение 0, определяет минимальное время паузы в миллисекундах между последовательными действиями компилятора по созданию цифровой подписи. Например, при значении 5000 компилятор не будет выполнять более одной подписи в течении 5 секунд. Может быть использовано для предотвращения конфликта доступа к службам создания отметки времени.
