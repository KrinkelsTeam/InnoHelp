[Setup]: SolidCompression

Допустимые значения:
yes или no

Значение по умолчанию:
no

Описание

При значении yes включается непрерывное сжатие данных. В результате все файлы будут сжаты в один непрерывный поток данных, а не раздельно. Это позволяет получить более высокую степень сжатия особенно, если установочные файлы содержат однотипные данные и будут сгруппированы в секции [Files] по типу.

Недостаток непрерывного сжатия заключается в том, что программа установки больше не сможет осуществлять случайный доступ к файлам установки. Это существенно снижает производительность инсталлятора. Чтобы, к примеру, извлечь конкретный файл, находящийся в середине потока, инсталлятору сначала придётся распаковать все файлы, которые находятся перед ним, и только потом извлечь нужный файл. Если при извлечении определённого файла произойдёт ошибка и пользователь нажмёт "Повторить", весь процесс распаковки начнётся с самого начала. А если ещё инсталлятор разбит по дискам, то пользователю придётся повторно вставить диск 1.
