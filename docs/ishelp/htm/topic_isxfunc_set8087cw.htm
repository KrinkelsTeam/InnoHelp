Pascal Scripting: Set8087CW


Прототип:

procedure Set8087CW(NewCW: Word);

Описание:

Устанавливает новое значение управляющего слова FPU и сохраняет его в переменной Default8087CW. Чаще всего функция Set8087CW служит для изменения точности операций с плавающей точкой, маски исключительных ситуаций и режима округления. Дополнительную информацию об управляющем состоянии FPU можно получить в документации по архитектуре процессоров Intel.

Пример:
Set8087CW($133f); // Отключает все исключения FPU
