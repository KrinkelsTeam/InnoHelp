Программирование: RegisterWindowMessage

Прототип

function RegisterWindowMessage(const Name: String): Longint;

Описание

Функция RegisterWindowMessage регистрирует новое сообщение окна, которое гарантированно будет уникальным во всей системе. Возвращаемое значение сообщения может быть использовано при вызове функции SendBroadcastMessage или PostBroadcastMessage.
