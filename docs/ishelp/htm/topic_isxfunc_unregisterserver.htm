Pascal Scripting: UnregisterServer


Прототип:

function UnregisterServer(const Is64Bit: Boolean; const Filename: String; const FailCriticalErrors: Boolean): Boolean;

Описание:

Выполняет отмену регистрации DLL/OCX с указанным именем файла. В случае успеха возвращает значение True, иначе False. На 64-бит системах, при условии что функция Is64Bit вернула True, файл DLL/OCX будет загружен как 64-бит образ и разрегистрирован в 64-бит процессе. Если параметр FailCriticalErrors установлен в True, то система не покажет ни одного сообщения об ошибке.
