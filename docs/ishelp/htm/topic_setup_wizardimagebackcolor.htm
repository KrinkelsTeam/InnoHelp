[Setup]: WizardImageBackColor


Описание:

Устаревшая директива с версии 5.5.7. Ранее эта директива определяла цвет фона, которым заполнялась вся неиспользуемая область вокруг изображения мастера установки, когда для WizardImageStretch было установлено значение no. Теперь всё неиспользуемое пространство заполняется стандартным цветом окна (обычно белым). Если вам необходимо создать рамку вокруг изображения другим цветом, воспользуйтесь графическим редактором и измените изображение.
