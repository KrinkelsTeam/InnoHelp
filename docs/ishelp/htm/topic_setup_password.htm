[Setup]: Password

Описание

Определяет пароль, который должен будет ввести пользователь, чтобы продолжить установку.

При использовании пароля также рекомендуется указать директиву Encryption со значением yes, в противном случае все файлы будут сохранены в текстовом виде и кому-то будет не так сложно получить к ним доступ, используя обратную разработку.

Сам пароль сохраняется в виде 160-разрядного хеша SHA1, модифицированного случайным 64-разрядным числом. (Примечание: при включённом шифровании этот хеш не используется для ключа шифрования, для этого создаётся другой хеш с другим модификатором.)
