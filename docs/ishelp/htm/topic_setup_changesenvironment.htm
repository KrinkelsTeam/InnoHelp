[Setup]: ChangesEnvironment


Допустимые значения:
yes или no, или логическое выражение

Значение по умолчанию:
no

Описание:

Если задано значение yes или логическое выражение вернёт значение True, по окончании инсталляции программа установки уведомит другие запущенные приложения (в частности Проводник Windows) о необходимости перезагрузки переменных среды из реестра.

Если при установке создаётся или изменяется переменная среды, но для директивы ChangesEnvironment не задано значение yes, новая или изменённая переменная среды не будет видна приложениями, запущенными из Проводника, до тех пор, пока пользователь не выйдет из системы или не перезагрузит компьютер.
