[Setup]: TouchDate


Допустимые значения:
current, none или YYYY-MM-DD

Значение по умолчанию:
current

Описание:

Определяет дату, используемую в отметке даты/времени для файлов секции [Files], у которых указан флаг touch.

Значение current устанавливает текущее системное время при компиляции программы установки. Значение none оставляет время без изменений. В противном случае значение директивы TouchDate интерпретируется явным образом согласно формата YYYY-MM-DD (ISO 8601). Если для TimeStampsInUTC указано значение yes, дата устанавливается в соответствии со стандартом UTC.

Пример:
TouchDate=2004-01-31
