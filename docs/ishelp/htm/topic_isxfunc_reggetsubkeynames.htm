Pascal Scripting: RegGetSubkeyNames


Прототип:

function RegGetSubkeyNames(const RootKey: Integer; const SubKeyName: String; var Names: TArrayOfString): Boolean;

Описание:

Открывает указанный ключ реестра и считывает имена подключей в заданный строковый массив Names. В случае успеха возвращает значение True, иначе False.

Пример:
var
  Names: TArrayOfString;
  I: Integer;
  S: String;
begin
  if RegGetSubkeyNames(HKEY_CURRENT_USER, 'Control Panel', Names) then
  begin
    S := '';
    for I := 0 to GetArrayLength(Names)-1 do
      S := S + Names[I] + #13#10;
    MsgBox('Список подключей:'#13#10#13#10 + S, mbInformation, MB_OK);
  end else
  begin
    // добавьте сюда код для обработки ошибки
  end;
end;
