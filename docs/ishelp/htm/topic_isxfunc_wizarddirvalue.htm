Pascal Scripting: WizardDirValue


Прототип:

function WizardDirValue: String;

Описание:

Возвращает текущее значение элемента управления ввода на странице мастера "Выбор каталога установки".

В отличие от ExpandConstant('{app}'), эта функция не завершится ошибкой, если будет вызвана после отображения страницы мастера, но перед тем, как пользователь выберет каталог. Вместо этого она вернёт имя каталога по умолчанию.
