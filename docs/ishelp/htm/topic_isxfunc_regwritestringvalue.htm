Pascal Scripting: RegWriteStringValue


Прототип:

function RegWriteStringValue(const RootKey: Integer; const SubKeyName, ValueName, Data: String): Boolean;

Описание:

Записывает в реестр указанное значение типа REG_SZ. В случае успеха возвращает значение True, иначе False.

Примечание:

Если значение уже существует и имеет тип REG_EXPAND_SZ, новое значение также будет иметь тип REG_EXPAND_SZ. В противном случае будет создано значение типа REG_SZ.

Пример:
begin
  RegWriteStringValue(HKEY_AUTO, 'Software\My Company\My Program',
    'Language', ExpandConstant('{language}'));
end;
