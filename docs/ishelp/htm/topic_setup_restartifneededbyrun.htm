[Setup]: RestartIfNeededByRun


Допустимые значения:
yes или no

Значение по умолчанию:
yes

Описание:

Эта директива в случае необходимости управляет перезагрузкой компьютера.

При значении yes и наличии у файлов секции [Files] флага restartreplace, который добавляет файлы в очередь на замену при следующей перезагрузке (путём вызова MoveFileEx или изменения wininit.ini), программа установки предложит пользователю выполнить перезагрузку компьютера по окончании установки.
