Pascal Scripting: SetNTFSCompression


Прототип:

function SetNTFSCompression(const FileOrDir: String; Compress: Boolean): Boolean;

Описание:

Изменяет состояние NTFS сжатия файла или каталога. В случае успеха возвращает значение True.

Примечание:

Если указан каталог, то состояние сжатия любых файлов, находящихся в этом каталоге, не будет изменено.
