[Setup]: PrivilegesRequired


Допустимые значения:
admin или lowest

Значение по умолчанию:
admin

Описание:

Эта директива управляет запросом повышения прав (посредством диалога управления учётными записями) при запуске программы установки.

При значении admin (по умолчанию) программа установки всегда будет выполняться с правами администратора и в режиме административной установки. Если программа установки была запущена непривилегированным пользователем, Windows запросит пароль учётной записи, имеющей права администратора, после чего программа установки продолжит работу под этой учётной записью.

При значении lowest программа установки не будет запрашивать права администратора, даже если была запущена пользователем из группы "Администраторы", и всегда будет выполняться в режиме не административной установки. Не используйте это значение, если не уверены, что установка будет успешно выполнена для непривилегированных учётных записей.
