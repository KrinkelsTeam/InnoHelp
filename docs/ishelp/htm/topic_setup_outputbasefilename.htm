[Setup]: OutputBaseFilename


Значение по умолчанию:
mysetup

Описание:

Эта директива позволяет указать другое имя для конечного файла программы установки, чтобы после компиляции его не приходилось переименовывать вручную.

Не рекомендуется использовать значение setup: все исполняемые файлы с именем "setup.exe" проверяются системой совместимости приложений Windows, которая подгружает дополнительные файлы библиотек, такие как version.dll. Загрузка этих файлов является небезопасной и может быть взломана.

Пример:
OutputBaseFilename=MyProg100
