Программирование: RegDeleteKeyIfEmpty

Прототип

function RegDeleteKeyIfEmpty(const RootKey: Integer; const SubkeyName: String): Boolean;

Описание

Выполняет удаление указанного ключа реестра, если только он не содержит подключей или значений. В случае успеха функция возвращает значение True, иначе — False.
