[Setup]: DefaultDirName


Описание:

Эта директива является обязательной и определяет имя каталога по умолчанию, в который будет выполнена установка приложения. Значение используется на странице "Выбор папки установки" и обычно начинается с константы.

Если директива UsePreviousAppDir имеет значение yes (используется по умолчанию) и программа установки обнаружит уже установленную предыдущую версию такого же приложения, вместо заданного по умолчанию каталога будет установлен выбранный ранее каталог.

Примеры:
Если задано:
DefaultDirName={sd}\MYPROG
Программа установки предложит путь:
C:\MYPROG

Если задано:
DefaultDirName={autopf}\My Program
Программа установки предложит путь:
C:\Program Files\My Program
