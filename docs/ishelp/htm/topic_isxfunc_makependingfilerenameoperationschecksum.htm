Pascal Scripting: MakePendingFileRenameOperationsChecksum


Прототип:

procedure MakePendingFileRenameOperationsChecksum: String;

Описание:

Возвращает контрольную сумму текущего значения реестра PendingFileRenameOperations. Вызывающая программа может использовать эту контрольную сумму для проверки изменений в PendingFileRenameOperations или файле WININIT.INI.

Пример:
var
  ChecksumBefore, ChecksumAfter: String;
begin
  ChecksumBefore := MakePendingFileRenameOperationsChecksum;
  // ...запуск программы...
  ChecksumAfter := MakePendingFileRenameOperationsChecksum;
  if ChecksumAfter <> ChecksumBefore then
    // PendingFileRenameOperations или WININIT.INI изменился
end;
