Pascal Scripting: RestartReplace


Прототип:

procedure RestartReplace(const TempFile, DestFile: String);

Описание:

Выполняет переименование TempFile в DestFile при следующем запуске Windows. Если DestFile уже существует, он будет перезаписан. Если DestFile не указан, то TempFile будет удалён. В случае ошибки будет вызвано исключение.
