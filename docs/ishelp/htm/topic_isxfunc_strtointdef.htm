Pascal Scripting: StrToIntDef

Прототип

function StrToIntDef(S: String; Def: LongInt): LongInt;

Описание

Преобразует числовую строку S в целочисленное значение и типом LongInt. Если результат преобразования строки S не содержит действительное значение, функция вернёт целое число, заданное в параметре Def.
