Pascal Scripting: FindWindowByWindowName

Прототип

function FindWindowByWindowName(const WindowName: String): HWND;

Описание

Получает дескриптор окна верхнего уровня, которое соответствует указанному имени окна WindowName. Эта функция не выполняет поиск дочерних окон и не учитывает регистр символов в имени окна. Возвращает значение 0, если окно не найдено.
