[Setup]: ArchitecturesAllowed


Допустимые значения:
Одно или несколько из следующих значений, разделённых пробелами:
x86
x64
arm64
ia64

Значение по умолчанию:
(пустое)

Описание:

Определяет архитектуру процессора, на которой разрешается запуск программы установки. Если директива не указана или является пустой, запуск программы установки разрешается на всех поддерживаемых архитектурах, способных выполнять 32-разрядный код установки (включая тот, который она не распознает). Иначе, если пользовательская архитектура не будет соответствовать одной из указанной, программа установки выдаст сообщение об ошибке и завершит свою работу.

Если все двоичные файлы вашего приложения являются 32-разрядными и выполняются в пользовательском режиме, не следует изменять значение этой директивы по умолчанию (в частности, не следует указывать значение x86), поскольку такие файлы без проблем работают в 64-разрядной Windows через эмулятор WOW64.

Если выполняется установка каких-либо 32-разрядных драйверов устройств, для этой директивы необходимо указать значение x86, т.к. 32-разрядные драйвера устройств не могут нормально работать в 64-разрядной Windows.

Если двоичные файлы вашего приложения созданы для архитектуры x64, ARM64 или Itanium, то для директивы необходимо указать значение x64, arm64 или ia64 соответственно.
