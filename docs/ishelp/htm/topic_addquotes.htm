Препроцессор Inno Setup: AddQuotes

Прототип функции

str AddQuotes(str S)

Описание

Заключает строку в кавычки ("), если она содержит пробел или ещё не была заключена в кавычки. В основном эту функцию можно использовать при создании другого процесса с длинным именем файла в качестве одного из параметров.
