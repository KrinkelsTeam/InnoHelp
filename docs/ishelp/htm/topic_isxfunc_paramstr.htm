Pascal Scripting: ParamStr


Прототип:

function ParamStr(Index: Integer): String;

Описание:

По заданному индексу Index возвращает один из параметров командной строки, переданных программе установки или удаления, за исключением недокументированных внутренних параметров, используемых ими.

Параметр с индексом 0 — это полное имя файла, с которого была запущена программа установки или удаления.

Если индекс недействительный, функция вернёт пустую строку.
