Препроцессор Inno Setup: GetPackedVersion

Прототип функции

str GetPackedVersion(str Filename, int *Version)

Описание

Получение упакованной версии заданного файла. В случае успеха возвращает строковое значение версии (в формате "0.0.0.0"), иначе — пустую строку.

Всегда используйте ComparePackedVersion или SamePackedVersion для сравнения упакованных версий.

Функция объявлена в файле ISPPBuiltins.iss.
