Pascal Scripting: ExtractFileName


Прототип:

function ExtractFileName(const FileName: string): String;

Описание:

Извлекает имя и расширение для указанного имени файла. Итоговая строка будет содержать все символы правой части FileName, начиная с первого символа после двоеточия или обратной косой черты, которые отделяют имя пути от имени файла с расширением. Если заданное имя не содержит определение пути (диск и каталог), то результатом будет исходное имя файла.
