Pascal Scripting: LoadDLL


Прототип:

function LoadDLL(const DLLName: String; var ErrorCode: Integer): Longint;

Описание:

Выполняет загрузку указанного файла DLL. Если файл был успешно загружен, то возвращает дескриптор DLL, в противном случае ноль. Если функция вернула ноль, то параметр ErrorCode укажет на причину ошибки. Для получения описания ошибки используйте вызов SysErrorMessage(ErrorCode).

Примечание:

Эта функция устарела. Смотрите раздел Использование DLL.

Смотрите коды системных ошибок на MSDN.
