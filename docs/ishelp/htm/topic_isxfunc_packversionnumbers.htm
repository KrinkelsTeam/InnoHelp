Pascal Scripting: PackVersionNumbers


Прототип:

function PackVersionNumbers(const VersionMS, VersionLS: Cardinal): Int64;

Описание:

Выполняет упаковку номеров версии в одно значение.

Для сравнения упакованных версий всегда используйте ComparePackedVersion или SamePackedVersion.
