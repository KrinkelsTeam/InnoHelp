Pascal Scripting: WizardIsTaskSelected


Прототип:

function WizardIsTaskSelected(const Tasks: String): Boolean;

Описание:

Возвращает значение True, если выбрана указанная задача. Несколько задач можно указать таким же образом, как и параметры Tasks.

Пример:
begin
  if WizardIsTaskSelected('desktopicon') then
    // задача 'desktopicon' успешно выбрана
end;
