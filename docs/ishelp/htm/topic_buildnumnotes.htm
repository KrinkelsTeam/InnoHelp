Использование номера сборки и/или пакета обновления


Версии системы, определённые в MinVersion и OnlyBelowVersion, могут также содержать номера сборки и/или версии пакета обновления.

Примеры:
5.0.2195
5.0sp4
5.0.2195sp4

Если номер сборки не указан или равен нулю, то программа установки не будет проверять номер сборки системы.

Если версия пакета обновления не указана или равна нулю, то программа установки не будет проверять версию пакета обновления системы.

Если версия пакета обновления указана, то программа установки будет сравнивать её с версией пакета обновления системы только в том случае, если указанный номер старшей и младшей версии полностью соответствует версии системы. Например, если для MinVersion указано значение 5.0sp4, то программа установки выполнить проверку только для SP4 на системе Windows 2000 (5.0).

В параметре OnlyBelowVersion: если указанная версия совпадает с версией системы, то обычно программа установки считает эту версию слишком высокой. Однако, если ещё указана и версия пакета обновления, то указанная версия может оказаться в пределах допуска версии системы. Например, на системе Windows 2000 SP4, значения 5.0 и 5.0.2195 не пройдут проверку OnlyBelowVersion, но со значениями 5.0sp5 и 5.0.2195sp5 проверка будет успешной (так как SP4 < sp5).
