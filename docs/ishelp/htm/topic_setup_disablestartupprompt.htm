[Setup]: DisableStartupPrompt

Допустимые значения:
yes или no

Значение по умолчанию:
yes

Описание

Эта директива управляет отображением сообщения с подтверждением установки.

Если задано значение yes, программа установки не будет показывать сообщение "Данная программа установит ... на ваш компьютер, продолжить?".

Эта директива не учитывается, если UseSetupLdr имеет значение no.
