Pascal Scripting: WizardGroupValue


Прототип:

function WizardGroupValue: String;

Описание:

Возвращает текущее значение элемента управления ввода на странице мастера Выбор папки в меню Пуск.

В отличие от ExpandConstant('{group}'), эта функция не завершится ошибкой, если будет вызвана после отображения страницы мастера, но перед тем, как пользователь выберет папку. Вместо этого она вернёт имя папки по умолчанию.
