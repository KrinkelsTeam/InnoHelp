Программирование: SendBroadcastMessage

Прототип

function SendBroadcastMessage(const Msg, WParam, LParam: Longint): Longint;

Описание

Отправляет указанное сообщение Msg с параметрами Wparam и Lparam окнам верхнего уровня в системе. Не возвращает результат, пока все оконные процедуры не обработают сообщение.

Указанное сообщение должно быть уникальным. Чтобы получить такое сообщение, используйте функцию RegisterWindowMessage.
