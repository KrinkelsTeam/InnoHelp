Pascal Scripting: SendBroadcastMessage


Прототип:

function SendBroadcastMessage(const Msg, WParam, LParam: Longint): Longint;

Описание:

Отправляет указанное сообщение окнам верхнего уровня в системе. Не возвращает результат, пока все оконные процедуры не обработают сообщение.

Указанное сообщение должно быть уникальным. Чтобы получить такое сообщение, используйте функцию RegisterWindowMessage.
