Pascal Scripting: RegQueryBinaryValue

Прототип

function RegQueryBinaryValue(const RootKey: Integer; const SubKeyName, ValueName: String; var ResultStr: AnsiString): Boolean;

Описание

Запрашивает в указанном разделе реестра значение с типом REG_BINARY и возвращает данные в параметре ResultStr. В случае успеха функция возвращает значение True. Если функция возвращает False, то данные ResultStr остаются без изменений.

Пример

var
  V: String;
begin
  if RegQueryStringValue(HKLM, 'SOFTWARE\Google\Google Sketchup 6', '', V) then
    MsgBox('Значение составляет "' + V + '"', mbInformation, MB_OK);
  Result := True;
end;
