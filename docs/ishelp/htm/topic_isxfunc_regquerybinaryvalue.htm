Pascal Scripting: RegQueryBinaryValue


Прототип:

function RegQueryBinaryValue(const RootKey: Integer; const SubKeyName, ValueName: String; var ResultStr: AnsiString): Boolean;

Описание:

Запрашивает значение типа REG_BINARY в указанном разделе реестра и возвращает данные в параметре ResultStr. В случае успеха возвращает значение True. Если функция возвращает False, то данные ResultStr остаются без изменений.
