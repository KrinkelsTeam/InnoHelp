Программирование: SetIniString

Прототип

function SetIniString(const Section, Key, Value, Filename: String): Boolean;

Описание

Выполняет запись строкового значения (тип String) в файл INI.

Параметр FileName определяет имя файла INI. Параметром Section задаётся имя раздела, а параметром Key — имя ключа. В параметре Value указывается желаемое значение.
