Pascal Scripting: CreateOutputMarqueeProgressPage


Прототип:

function CreateOutputMarqueeProgressPage(const ACaption, ADescription: String): TOutputMarqueeProgressWizardPage;

Описание:

Выполняет создание страницы мастера, которая содержит статичный текст и непрерывный (стиль Marquee) индикатор выполнения.

Информацию о создании страниц с индикатором выполнения смотрите в разделе CreateOutputProgressPage.

Примечание:

Для анимации индикатора выполнения используйте вызов метода Animate. Не используйте вызов метода SetProgress, это приведёт к внутренней ошибке.

Смотрите пример в файле AllPagesExample.iss.
