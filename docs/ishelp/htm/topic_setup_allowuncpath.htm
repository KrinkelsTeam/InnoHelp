[Setup]: AllowUNCPath


Допустимые значения:
yes или no

Значение по умолчанию:
yes

Описание:

Если директива имеет значение no, то пользователю не будет разрешено вводить UNC-путь (такой как "\\server\share") на странице "Выбор папки установки". Такое поведение было по умолчанию в Inno Setup 2.0.17 и ранее.

Чтобы полностью запретить установку в сетевые папки, следует также для директивы AllowNetworkDrive указать значение no.
