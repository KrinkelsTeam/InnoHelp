[Setup]: LZMAMatchFinder


Допустимые значения:
HC или BT

Значение по умолчанию:
HC, если уровень сжатия LZMA равен fast
иначе BT

Описание:

Эта директива управляет методом поиска совпадений, используемого компрессорами LZMA и LZMA2.

Значение HC активирует метод Hash Chain с 4 хеш-байтами.

Значение BT активирует метод Binary Tree с 4 хеш-байтами.

Метод Binary Tree может обеспечить лучшую степень сжатия за счёт скорости сжатия.
