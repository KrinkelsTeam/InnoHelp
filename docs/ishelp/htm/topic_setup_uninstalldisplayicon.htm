[Setup]: UninstallDisplayIcon


Описание:

Эта директива позволяет указать другой файл значка приложения, который будет отображаться в апплете панели управления "Удаление или изменение программы". В качестве значка может быть использован исполняемый файл или файл *.ico. Имя файла обычно начинается с константы каталога.

Если выбранный файл содержит несколько значков, можно добавить суффикс ",n", где n – указывает индекс значка (отсчёт индексов начинается с нуля и он используется по умолчанию).

Если директива не задана или пустая, Windows самостоятельно выберет значок, который может оказаться не тем, который вы хотели бы видеть.

Примеры:
UninstallDisplayIcon={app}\MyProg.exe
UninstallDisplayIcon={app}\MyProg.exe,1
