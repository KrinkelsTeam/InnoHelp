[Setup]: TouchTime


Допустимые значения:
current, none, HH:MM или HH:MM:SS

Значение по умолчанию:
current

Описание:

Определяет время, используемое в отметке даты/времени для файлов секции [Files], у которых указан флаг touch.

Значение current устанавливает текущее системное время при компиляции программы установки. Значение none оставляет время без изменений. В противном случае значение директивы TouchTime интерпретируется явным образом согласно формата HH:MM или HH:MM:SS. Если для TimeStampsInUTC указано значение yes, время устанавливается в соответствии со стандартом UTC.

Пример:
TouchTime=13:00
