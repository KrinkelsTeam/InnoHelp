[Setup]: WindowShowCaption

Допустимые значения:
yes или no

Значение по умолчанию:
yes

Описание

Если указано значение no, программа установки будет действительно "полноэкранной" — без заголовка и границ, а также будет размещаться поверх всех окон на панели задач.

Этот параметр не даёт никакого эффекта, если директиве WindowVisible не задано значение yes.
