[Setup]: PrivilegesRequiredOverridesAllowed


Допустимые значения:
Одно или более значений, разделённых пробелами:
commandline
dialog

Значение по умолчанию:
(пустое)

Описание:

Данная директива позволяет конечному пользователю переопределить используемое в сценарии значение PrivilegesRequired.

При значении commandline в программе установки активируется поддержка дополнительных параметров командной строки /ALLUSERS и /CURRENTUSER, которые позволяют переопределить значение директивы PrivilegesRequired. Более подробную информацию смотрите в разделе справки "Параметры командной строки инсталлятора".

При значении dialog программа установки предложит пользователю выбрать режим установки на основе указанного в сценарии значения PrivilegesRequired, используя подавляемое диалоговое окно. Также значение dialog автоматически активирует commandline, и если будет использован один из параметров командной строки, программа установки не будет показывать этот диалог.
