[Setup]: DisableFinishedPage


Допустимые значения:
yes или no

Значение по умолчанию:
no

Описание:

Эта директива управляет отображением страницы завершения установки.

Если задано значение yes, программа установки не будет показывать страницу "Установка завершена", а вместо этого будет немедленно закрыта сразу же по завершении процесса установки. Директива может быть использована, когда для запуска программы в секции [Run] указан флаг nowait, и нет никакой необходимости оставлять окно "Установка завершена" в фоновом режиме после запуска другой программы.

Обратите внимание, что директива DisableFinishedPage будет пропущена, если по условиям установки необходимо выполнить перезагрузку компьютера, или для директивы InfoAfterFile в секции [Setup] был назначен файл. В таких случаях страница мастера "Установка завершена" будет все равно показана.
