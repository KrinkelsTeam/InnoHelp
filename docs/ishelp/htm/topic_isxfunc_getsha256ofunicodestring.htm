Pascal Scripting: GetSHA256OfUnicodeString


Прототип:

function GetSHA256OfUnicodeString(const S: String): String;

Описание:

Возвращает строковое значение хеш-суммы SHA-256 для указанной строки в кодировке Unicode.

Пример:
var
  SHA256: String;
begin
  SHA256 := GetSHA256OfUnicodeString('Test');
  // SHA256 = 'fe520676b1a1d93dabab2319eea03674f3632eaeeb163d1e88244f5eb1de10eb'
end;
