Программирование: GetSHA256OfUnicodeString

Прототип

function GetSHA256OfUnicodeString(const S: String): String;

Описание

Возвращает строковое значение контрольной суммы SHA-256 для заданной строки в кодировке Unicode.

Пример

var
  SHA256: String;
begin
  SHA256 := GetSHA256OfUnicodeString('Test');
  // SHA256 = 'e6fa3ca87b1b641ab646d3b4933bba8d0970763f030b6578a60abdeae7366247'
end;
