Программирование: AnsiUppercase

Прототип

function AnsiUppercase(S: AnyString): String;

Описание

Возвращает копию заданной строки S, в которой все символы преобразованы в верхний регистр. Преобразование использует текущие языковые настройки Windows. Эта функция поддерживает многобайтовые наборы символов (MBCS).
