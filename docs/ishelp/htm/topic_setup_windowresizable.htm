[Setup]: WindowResizable

Допустимые значения:
yes или no

Значение по умолчанию:
yes

Описание

При значении no пользователь не сможет изменить размер фонового окна программы установки, когда оно не развернуто.

Этот параметр не даёт никакого эффекта, если директиве WindowVisible не задано значение yes.
