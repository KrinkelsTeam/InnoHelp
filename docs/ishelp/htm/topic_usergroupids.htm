Идентификаторы пользователей и групп

admins Группа "Администраторы"
authusers Группа пользователей, прошедших проверку
creatorowner Создатель-владелец
everyone Группа "Все"
guests Группа "Гости"
networkservice Учётная запись сетевой службы
service Учётная запись локальной службы
system Учётная запись локальной системы
users Группа "Пользователи"
