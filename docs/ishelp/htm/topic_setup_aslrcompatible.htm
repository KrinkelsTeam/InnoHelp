[Setup]: ASLRCompatible

Допустимые значения:
yes или no

Значение по умолчанию:
yes

Описание

Данная директива указывает компилятору на необходимость установки флага "Dynamic Base" в заголовках исполняемых файлов программы установки и удаления. Эта функция появилась в Inno Setup 5.5.7 и по умолчанию имеет значение yes; предыдущие версии Inno Setup никогда не устанавливали этот флаг.
