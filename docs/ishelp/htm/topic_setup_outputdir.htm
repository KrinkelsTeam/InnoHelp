[Setup]: OutputDir


Значение по умолчанию:
Output

Описание:

Определяет для сценария каталог "вывода", в котором компилятор будет создавать файлы установки SETUP.*. По умолчанию вывод осуществляется в папку с именем "Output", которая создаётся в одном каталоге с файлом сценария.

Если OutputDir не содержит полного пути, то он будет создан относительно пути в параметре SourceDir. Если же имя пути содержит приставку "userdocs:", то вывод будет выполнен относительно папки "Мои документы" для текущего пользователя, который вошёл в систему. Установка в директиве OutputDir значения . приведёт к выводу файлов установки в исходный каталог сценария.

Пример:
OutputDir=c:\output
