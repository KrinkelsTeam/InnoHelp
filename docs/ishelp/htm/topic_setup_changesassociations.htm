[Setup]: ChangesAssociations


Допустимые значения:
yes или no, или логическое выражение

Значение по умолчанию:
no

Описание:

Если задано значение yes или логическое выражение вернёт значение True, по окончании инсталляции программа установки укажет Проводнику выполнить обновление информации об ассоциации файлов. Тоже самое сделает программа удаления в конце деинсталляции.

Если при установке создаётся файловая ассоциация, но для директивы ChangesAssociations не задано значение yes, то, скорее всего, корректный значок, связанный с типом файла, не будет отображаться до тех пор, пока пользователь не выйдет из системы или не перезагрузит компьютер.
