Pascal Scripting: SetIniInt

Прототип

function SetIniInt(const Section, Key: String; const Value: LongInt; const Filename: String): Boolean;

Описание

Выполняет запись целочисленного значения (тип LongInt) в файл INI.

Параметр FileName определяет имя файла INI. Параметром Section задаётся имя раздела, а параметром Key — имя ключа. В параметре Value указывается желаемое значение.
