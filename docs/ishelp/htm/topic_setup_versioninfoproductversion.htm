[Setup]: VersionInfoProductVersion


Допустимые значения:
Значение в виде 4-х чисел, разделённых точкой

Значение по умолчанию:
VersionInfoVersion

Описание:

Определяет двоичное значение версии продукта для сведений о версии программы установки.

Допускается указывать частичные номера версий. Пропущенные числа будут добавлены в виде нулей.

Обратите внимание, что это значение отображается только в проводнике Windows Vista SP2. В других версиях отображается текстовое значение версии продукта (VersionInfoProductTextVersion).
