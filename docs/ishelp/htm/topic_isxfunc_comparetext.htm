Pascal Scripting: CompareText

Прототип

function CompareText(const S1, S2: string): Integer;

Описание

Выполняет сравнение строк S1 со строками S2 без учёта регистра символов и возвращает значение 0, если строки равны. Если S1 больше чем S2, то функция возвращает целое значение больше 0. Если же S1 меньше чем S2, то функция вернёт целое значение меньше 0.
