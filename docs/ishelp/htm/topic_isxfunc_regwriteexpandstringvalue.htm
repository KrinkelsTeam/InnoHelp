Pascal Scripting: RegWriteExpandStringValue


Прототип:

function RegWriteExpandStringValue(const RootKey: Integer; const SubKeyName, ValueName, Data: String): Boolean;

Описание:

Записывает в реестр указанное значение типа REG_EXPAND_SZ. В случае успеха возвращает значение True, иначе False.

Пример:
begin
  RegWriteStringValue(HKEY_CURRENT_USER, 'Software\My Company\My Program',
    'UserName', '%UserName%');
end;
