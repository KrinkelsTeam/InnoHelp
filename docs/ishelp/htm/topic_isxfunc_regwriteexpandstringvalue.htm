Программирование: RegWriteExpandStringValue

Прототип

function RegWriteExpandStringValue(const RootKey: Integer; const SubKeyName, ValueName, Data: String): Boolean;

Описание

Записывает в реестр указанное значение с типом REG_EXPAND_SZ. В случае успеха функция возвращает значение True, иначе — False.

Пример

begin
  RegWriteStringValue(HKEY_CURRENT_USER, 'Software\My Company\My Program',
    'UserName', '%UserName%');
end;
