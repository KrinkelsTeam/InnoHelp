Pascal Scripting: ScaleX


Прототип:

function ScaleX(X: Integer): Integer;

Описание:

Возвращает заданное значение координаты X или ширины в масштабе, соответствующем размеру шрифта текущего диалога. Если диалог использует шрифт MS Sans Serif размером 8 точек и пользователь работает под управлением Windows с разрешением 96 dpi, то функция вернёт X без изменений.
