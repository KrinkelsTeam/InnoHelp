[Setup]: AppendDefaultGroupName

Допустимые значения:
yes или no

Значение по умолчанию:
yes

Описание

По умолчанию, когда пользователь выберет папку в диалоговом окне, который отображается при нажатии кнопки Обзор... на странице "Выбор папки в меню «Пуск»", программа установки автоматически добавит в новый путь имя папки из директивы DefaultGroupName. Например, если директива DefaultGroupName имеет значение My Program и пользователь выберет каталог "Accessories", новый путь примет вид "Accessories\My Program".

Установка для этой директивы значения no отключает добавление папки из параметра DefaultGroupName в новый путь. Кроме того, в диалоговом окне выбора папки появится кнопка Создать папку.
