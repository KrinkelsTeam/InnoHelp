[Setup]: UninstallDisplaySize


Описание:

В ОС Windows 7 и выше программа установки использует эту директиву для записи значения EstimatedSize в разделе реестра деинсталляции приложения, поскольку Windows 7 больше не вычисляет его автоматически (см. апплет "Программы и компоненты", раздел "Удаление или изменение программы"). Если директива UninstallDisplaySize не задана, программа установки оценивает примерный размер приложения на основании всех установленных файлов и добавляет все значения в ExtraDiskSpaceRequired. Примечание: Windows 7 без пакета обновления поддерживает отображение значений не более 4 Гб.

Значение необходимо указывать в байтах (1048576 байт = 1 Мб).

Пример:
UninstallDisplaySize=1073741824
