Pascal Scripting: CreateOutputMsgPage


Прототип:

function CreateOutputMsgPage(const AfterID: Integer; const ACaption, ADescription, AMsg: String): TOutputMsgWizardPage;

Описание:

Выполняет создание страницы мастера, которая содержит только статичный текст. Параметр AMsg определяет текст для отображения.

Пример:
var
  Page: TOutputMsgWizardPage;

...

// Создаём страницу
Page := CreateOutputMsgPage(wpWelcome,
  'Информация', 'Пожалуйста, ознакомьтесь с важной информацией.',
  'Бла-бла-бла.');

См. другой пример в файле AllPagesExample.iss.
