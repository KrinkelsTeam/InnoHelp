Pascal Scripting: GetOpenFileName


Прототип:

function GetOpenFileName(const Prompt: String; var FileName: String; const InitialDirectory, Filter, DefaultExtension: String): Boolean;

Описание:

Выполняет вызов диалогового окна, в котором пользователь может выбрать существующий файл. Возвращает значение True, если пользователь подтвердил выбор файла, иначе False. Имя выбранного файла передаётся параметру FileName.

Примечание:

К примеру, фильтр файлов: 'Text files (*.txt)|*.txt|All files (*.*)|*.*'

Пример:
var
  FileName: String;
begin
  // Устанавливаем начальное имя файла
  FileName := '';
  if GetOpenFileName('', FileName, '',
     'Текстовые файлы (*.txt)|*.txt|Все файлы|*.*', 'txt') then
  begin
    // Успех: пользователь нажал OK
    // Переменная FileName содержит выбранное имя файла
  end;
end;
