[Setup]: DisableDirPage


Допустимые значения:
auto, yes или no

Значение по умолчанию:
auto

Описание:

Эта директива управляет отображением страницы с выбором папки установки. Если задано значение yes, программа установки не будет показывать страницу "Выбор папки установки".

При значении auto, если программа установки обнаружит уже установленную версию такого же приложения, то страница "Выбор папки установки" показана не будет.

Если страница "Выбор папки установки" не отображается, то для установки всегда будет использоваться имя каталога по умолчанию.

См. также AlwaysShowDirOnReadyPage.
