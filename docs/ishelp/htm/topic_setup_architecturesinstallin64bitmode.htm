[Setup]: ArchitecturesInstallIn64BitMode


Допустимые значения:
Одно или несколько из следующих значений, разделённых пробелами:
x64
arm64
ia64

Значение по умолчанию:
(пустое)

Описание:

Определяет 64-разрядную архитектуру процессора, на которой должна работать программа установки в 64-разрядном режиме установки. Если директива не указана или является пустой, программа установки всегда будет работать в 32-разрядном режиме установки.

Обычно нет никакой необходимости изменять значение этой директивы по умолчанию, если только приложение не содержит 64-разрядные двоичные файлы.

Прежде чем использовать эту директиву, ознакомьтесь с разделом "Ограничения в 64-разрядном режиме".

Если ваше приложение работает только на 64-разрядных архитектурах процессора, следует в директиве ArchitecturesAllowed указать такое же самое значение, как и для этой директивы, чтобы предотвратить работу программы установки на 32-разрядных Windows.
