[Setup]: DiskSpanning


Допустимые значения:
yes или no

Значение по умолчанию:
no

Описание:

При установке значения yes активируется функция разделения данных. Чтобы не объединять все данные в один сжатый файл SETUP.EXE, компилятор будет их разбивать на несколько отдельных файлов SETUP-*.BIN — именуемых "сегментами" — которые потом можно скопировать на дискеты, CD или DVD диски. Каждый созданный сегмент будет содержать в своем имени число, указывающее порядковый номер диска, на который он должен быть скопирован. (Например, файл SETUP-2.BIN следует записать на диск 2.) Созданный файл SETUP.EXE всегда следует записывать на диск 1 вместе с файлом SETUP-1*.BIN.

Размер каждого сегмента и их количество для каждого диска определяются значениями директив DiskSliceSize и SlicesPerDisk в секции [Setup]. К другим директивам, связанными с дисками, которые можно изменить, относятся DiskClusterSize и ReserveBytes.

Обратите внимание, что этой директиве обязательно следует присвоить значение yes, если упакованный размер инсталлятора превышает размер 2,100,000,000 байт, даже если разделение установки на диски не планируется. (Установка по-прежнему будет работать корректно, если все файлы SETUP-*.BIN будут размещены на одном носителе.)
