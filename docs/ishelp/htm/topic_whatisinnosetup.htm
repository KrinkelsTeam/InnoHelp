Домашняя страница Inno Setup


Inno Setup - это бесплатный установщик для программ Windows, разработанный Jordan Russell и Martijn Laan. Первая версия этой программы появилась в 1997 году и сегодня Inno Setup является одной из самой популярной по созданию программ установок и даже превосходит многие коммерческие продукты по набору функций и стабильности.


Ключевые особенности:

  • Поддержка каждого выпуска Windows, начиная с 2006 года, включая: Windows 10, Windows 10 на базе ARM, Windows Server 2019, Windows Server 2016, Windows 8.1, Windows 8, Windows Server 2012, Windows 7, Windows Server 2008 R2, Windows Server 2008 и Windows Vista. (Пакеты обновлений не требуются.)
  • Обширная поддержка установки 64-разрядных приложений на 64-разрядные версии Windows. Также поддерживаются архитектуры x64, ARM64 и Itanium.
  • Обширная поддержка как для административных, так и для не административных установок.
  • Поддержка создания одного EXE-файла программы установки для удобного распространения в сети Интернет. Также поддерживается разбивка на диски.
  • Изменяемый размер стандартного окна мастера установки.
  • Создание настраиваемого типа установки, например: Полная, Минимальная, Выборочная.
  • Возможность создания деинсталлятора (программы удаления).
  • Установка файлов:
    Включает интегрированную поддержку для "deflate", bzip2, and 7-Zip LZMA/LZMA2 методов сжатия файлов. Программа установки также может сравнивать версии файлов, выполнять замену используемых/занятых файлов, назначать применение общих файлов, регистрировать DLL/OCX и другие типы библиотек, устанавливать шрифты.
  • Создание ярлыков в любом месте, в том числе в меню "Пуск" и на Рабочем столе.
  • Создание записей в реестре и файлах INI.
  • Возможность запуска сторонних программ перед началом, в течении и после окончания установки.
  • Поддержка многоязычных программ установок, в том числе с языком справа налево.
  • Поддержка защищенных паролем и шифрованных программ установок.
  • Создание программ установок и удаления с цифровой подписью, включая двойную подпись (SHA1 и SHA256).
  • Тихая установка и тихое удаление.
  • Unicode версия программ установок.
  • Встроенная функция предварительной обработки сценариев (препроцессор).
  • Встроенный компилятор сценариев на языке Pascal, позволяющего расширить функции и возможности программ установок и удаления.
  • Полностью открытый исходный код, доступный на ресурсе GitHub.
  • Небольшой размер модуля установки: всего лишь 1,5 Мб со всеми функциями.
  • Все функции полностью документированы.
  • Используется Microsoft Visual Studio Code и Embarcardero Delphi.


Inno Setup действительно полностью бесплатная программа, даже для коммерческого применения?

Да, Inno Setup может использоваться абсолютно бесплатно, даже для развёртывания коммерческих приложений.
