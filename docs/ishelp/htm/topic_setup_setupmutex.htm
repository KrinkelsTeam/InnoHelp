[Setup]: SetupMutex


Описание:

Эта директива используется для предотвращения повторного запуска программы установки во время её работы. Он определяет один или несколько именованных мьютексов (при указании нескольких мьютексов их необходимо разделять запятыми), которые программа установки будет проверять при запуске. Если хоть один из них существует, будет выдано сообщение: "Обнаружен запущенный экземпляр программы установки. Пожалуйста, закройте все экземпляры приложения, затем нажмите «OK», чтобы продолжить, или «Отмена», чтобы выйти.". Если ни один из них не обнаружен, программа установки создаст мьютекс(ы) и продолжит свою работу в обычном режиме. Значение может содержать константы.

Чтобы указать мьютекс с именем, содержащим запятую, экранируйте запятую обратной косой чертой.

Дополнительные сведения о мьютексах смотрите в разделе "CreateMutex" справки MS SDK.

Пример:
SetupMutex=MySetupsMutexName,Global\MySetupsMutexName
