Pascal Scripting: Copy


Прототип:

function Copy(S: String; Index, Count: Integer): String;

Описание:

Возвращает строку, которая содержит заданное количество символов Count из указанной строки S, начиная с позиции Index.

Если значение Index больше, чем длина строки S, то функция вернёт пустую строку.

Если значение Count определяет бо́льшее количество символов, чем доступно с строке S с указанной позиции Index, то функция вернёт только имеющиеся до конца строки символы.
