[Setup]: ShowLanguageDialog


Допустимые значения:
yes, no или auto

Значение по умолчанию:
yes

Описание:

При значении yes и наличии нескольких записей в секции [Languages], будет показан диалог "Выбор языка", в котором пользователь сможет переопределить заданный по умолчанию язык установки. Более подробную информацию смотрите в разделе справки Секция [Languages].

При значении no диалог никогда не будет показан.

При значении auto диалог будет показан только в том случае, если программа установки не найдёт соответствующий языковой идентификатор.
