[Setup]: AppCopyright


Описание:

Определяет сообщение об авторских правах, которое программа установки будет показывать в правом нижнем углу фонового окна установки, если директива WindowVisible имеет значение yes.

Значение этой директивы также используется в качестве значения по умолчанию для директивы VersionInfoCopyright, если оно не указано.

Пример:
AppCopyright=Copyright (C) 1997-2020 My Company, Inc.
