[Setup]: UninstallFilesDir


Значение по умолчанию:
{app}

Описание:

Определяет каталог для записи файлов деинсталляции "unins*.*".

Замечание: здесь не следует указывать различные значения для новых версий приложения, т.к. программа установки не сможет найти журналы деинсталляции от предыдущих версии и не сможет их дополнить.

Пример:
UninstallFilesDir={app}\uninst
