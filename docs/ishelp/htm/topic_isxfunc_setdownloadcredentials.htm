Pascal Scripting: SetDownloadCredentials


Прототип:

procedure SetDownloadCredentials(const User, Pass: String);

Описание:

Устанавливает имя пользователя и пароль для всех последующих загрузок. Для удаления предыдущего значения укажите пустую строку.

См. также:

DownloadTemporaryFile
DownloadTemporaryFileSize
DownloadTemporaryFileDate
