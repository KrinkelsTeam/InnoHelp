Программирование: SetDownloadCredentials

Прототип

procedure SetDownloadCredentials(const User, Pass: String);

Описание

Устанавливает имя пользователя User и пароль Pass для всех последующих загрузок. Для удаления предыдущего значения укажите в параметрах пустые строки.

См. также:

DownloadTemporaryFile
DownloadTemporaryFileSize
DownloadTemporaryFileDate
