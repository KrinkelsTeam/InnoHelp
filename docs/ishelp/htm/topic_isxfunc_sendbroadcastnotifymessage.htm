Pascal Scripting: SendBroadcastNotifyMessage

Прототип

function SendBroadcastNotifyMessage(const Msg, WParam, LParam: Longint): Boolean;

Описание

Отправляет указанное сообщение Msg с параметрами Wparam и Lparam окнам верхнего уровня без ожидания обработки сообщения оконными процедурами. В случае успеха функция возвращает значение True.
