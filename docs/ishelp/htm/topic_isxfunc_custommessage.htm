Pascal Scripting: CustomMessage


Прототип:

function CustomMessage(const MsgName: String): String;

Описание:

Возвращает значение записи [CustomMessages] для указанного имени. Если запись с указанным именем не существует, будет создано исключение.

Пример:
var
  S: String;
begin
  S := CustomMessage('CreateDesktopIcon');
  // S = 'Создать ярлык на Рабочем столе'

  S := FmtMessage(CustomMessage('NameAndVersion'), ['My Program', '1.0']);
  // S = 'My Program version 1.0'
end;
