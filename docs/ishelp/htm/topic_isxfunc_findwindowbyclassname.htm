Программирование: FindWindowByClassName

Прототип

function FindWindowByClassName(const ClassName: String): HWND;

Описание

Получает дескриптор окна верхнего уровня, которое соответствует указанному имени класса ClassName. Эта функция не выполняет поиск дочерних окон и не учитывает регистр символов в имени класса. Возвращает значение 0, если окно не найдено.
