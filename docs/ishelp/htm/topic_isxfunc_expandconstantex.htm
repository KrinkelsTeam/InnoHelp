Программирование: ExpandConstantEx

Прототип

function ExpandConstantEx(const S: String; const CustomConst, CustomValue: String): String;

Описание

Заменяет все константы в строке S на соответствующие им значения. Кроме того, любая константа, равная CustomConst, будет заменена на CustomValue.

Если при расширении константы произойдет ошибка, будет вызвано исключение.
