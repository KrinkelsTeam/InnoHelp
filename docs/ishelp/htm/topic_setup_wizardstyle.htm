[Setup]: WizardStyle

Допустимые значения:
classic или modern

Значение по умолчанию:
classic

Описание

Эта директива позволяет изменить внешний вид программы установки/удаления. При указании значения modern, инсталлятор примет более современный вид, а значения по умолчанию для директив WizardResizable и WizardSizePercent будут изменены на новые yes и 120,120 соответственно.

Чтобы добавить полную поддержку WizardResizable и WizardSizePercent для всех пользовательских элементов управления и страниц мастера, если таковые имеются, используйте свойства Anchors и KeepSizeY. В качестве примера смотрите файл сценария CodeClasses.iss.
