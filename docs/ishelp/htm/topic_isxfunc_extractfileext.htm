Pascal Scripting: ExtractFileExt

Прототип

function ExtractFileExt(const FileName: String): String;

Описание

Возвращает расширение для указанного имени файла, переданного в параметре FileName. Итоговая строка включает символ точки, которая разделяет имя и расширение файла. Если заданное имя файла не имеет расширения, результатом функции будет пустая строка.
