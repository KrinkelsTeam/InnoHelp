Pascal Scripting: GetTempDir


Прототип:

function GetTempDir: String;

Описание:

Возвращает полный путь к временному каталогу, при этом строка пути будет содержать завершающий символ разделения папок (обратная косая черта). Из-за различия в платформах функция API Win32 GetTempPath не используется.

  Поиск и получение пути выполняется в следующем порядке:

  1. Из переменной среды TMP.
  2. Из переменной среды TEMP, если переменная TMP не определена или каталог, указанный в ней, не существует.
  3. Из переменной среды USERPROFILE, если обе переменные TMP и TEMP не определены или каталоги, указанные в них, не существуют.
  4. Из каталога Windows, если переменные TMP, TEMP и USERPROFILE не определены или сами каталоги не существуют.
