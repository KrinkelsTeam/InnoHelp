Pascal Scripting: CompareStr

Прототип

function CompareStr(const S1, S2: string): Integer;

Описание

Выполняет сравнение строки S1 со строкой S2 с учётом регистра символов.

  Возвращает одно из следующих значений:
   < 0 — если S1 меньше чем S2;
      0 — если строки S1 и S2 равны;
   > 0 — если S1 больше чем S2.
