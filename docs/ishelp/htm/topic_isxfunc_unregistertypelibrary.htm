Pascal Scripting: UnregisterTypeLibrary


Прототип:

function UnregisterTypeLibrary(const Is64Bit: Boolean; const Filename: String): Boolean

Описание:

Выполняет отмену регистрации библиотеки типов с указанным именем файла. В случае успеха возвращает значение True, иначе False. На 64-бит системах, при условии что функция Is64Bit вернула True, будет отменена регистрация только в 64-бит процессе.
