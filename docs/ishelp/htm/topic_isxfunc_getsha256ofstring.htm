Pascal Scripting: GetSHA256OfString

Прототип

function GetSHA256OfString(const S: AnsiString): String;

Описание

Возвращает строковое значение контрольной суммы SHA-256 для заданной строки в кодировке ANSI.

Пример

var
  SHA256: String;
begin
  SHA256 := GetSHA256OfString('Test');
  // SHA256 = '9f86d081884c7d659a2feaa0c55ad015a3bf4f1b2b0b822cd15d6c15b0f00a08'
end;
