Pascal Scripting: ScaleY


Прототип:

function ScaleY(Y: Integer): Integer;

Описание:

Возвращает заданное значение координаты Y или высоты в масштабе, соответствующем размеру шрифта текущего диалога. Если диалог использует шрифт MS Sans Serif размером 8 точек и пользователь работает под управлением Windows с разрешением 96 dpi, то функция вернёт Y без изменений.
