Pascal Scripting: AddQuotes


Прототип:

function AddQuotes(const S: String): String;

Описание:

Добавляет символ кавычки (") с левой и правой стороны строки, если только строка не является пустой или уже не заключена в кавычки. Используется прежде всего при создании другого процесса, имеющем длинное имя в качестве одного из параметров.
