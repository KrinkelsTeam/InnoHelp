Pascal Scripting: WizardSelectTasks


Прототип:

procedure WizardSelectTasks(const Tasks: String);

Описание:

Выполняет выбор указанных в списке задач, разделённых запятыми.

Если имя задачи начинается символом "!", то её выбор будет отменён.

Эта функция не изменяет состояние задач, которых нет в списке.

Пример:
begin
  WizardSelectTasks('!desktopicon');
  // выбор задачи 'desktopicon' отменён
end;
