Pascal Scripting: RegQueryMultiStringValue


Прототип:

function RegQueryMultiStringValue(const RootKey: Integer; const SubKeyName, ValueName: String; var ResultStr: String): Boolean;

Описание:

Запрашивает значение типа REG_MULTI_SZ в указанном разделе реестра и возвращает данные в параметре ResultStr. В случае успеха возвращает значение True. Если функция возвращает False, то данные ResultStr остаются без изменений.

Примечание:

В значении типа REG_MULTI_SZ каждая строка разделяется нулевым символом (#0).
