Pascal Scripting: CreateOutputMsgMemoPage


Прототип:

function CreateOutputMsgMemoPage(const AfterID: Integer; const ACaption, ADescription, ASubCaption: String; const AMsg: AnsiString): TOutputMsgMemoWizardPage;

Описание:

Выполняет создание на странице мастера статического текста, доступного только для чтения, а также многострочного текстового редактора, способного отобразить документ в формате RTF. Параметр ASubCaption определяет статический текст для отображения. Параметр AMsg определяет текст, который будет назначен элементу редактирования.

Пример:
var
  Page: TOutputMsgMemoWizardPage;

...

// Создаём страницу
Page := CreateOutputMsgMemoPage(wpWelcome,
  'Информация', 'Пожалуйста, ознакомьтесь с важной информацией.',
  'Когда вы будете готовы продолжить установку, нажмите "Далее".',
  'Бла-бла-бла.');

См. другой пример в файле AllPagesExample.iss.
