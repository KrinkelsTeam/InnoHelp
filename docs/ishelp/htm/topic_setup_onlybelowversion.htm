[Setup]: OnlyBelowVersion

Формат:
major.minor

Значение по умолчанию:
0 или больше чем 6.1

Описание

Эта директива позволяет указать минимальную версию Windows, в которой программное обеспечение не будет работать. Значение 0 отключает верхнее ограничения версии. Также могут быть указаны номера сборки и/или пакеты обновления.

Эта директива по сути является противоположностью MinVersion.
