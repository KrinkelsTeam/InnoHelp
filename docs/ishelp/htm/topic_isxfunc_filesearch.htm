Pascal Scripting: FileSearch


Прототип:

function FileSearch(const Name, DirList: string): String;

Описание:

Выполняет поиск указанного имени файла Name в заданных каталогах DirList. Если каталогов несколько, то их имена должны быть отделены друг от друга точкой с запятой. Если функция находит соответствующий файл, то она возвращает полный путь к этому файлу. Если совпадений не найдено, возвращается пустая строка.
