Программирование: ForceDirectories

Прототип

function ForceDirectories(Dir: String): Boolean;

Описание

Выполняет создание одно и более вложенных каталогов по указанному пути Dir. Если первые каталоги в пути существуют, а последние нет, то функция создаст только те, которые не существуют. В случае успеха функция возвращает значение True, иначе False.
