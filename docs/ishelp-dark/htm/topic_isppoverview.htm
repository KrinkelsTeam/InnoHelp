Препроцессор Inno Setup: Введение

Препроцессор Inno Setup (ISPP) - это программа предварительной обработки сценариев для Inno Setup.

Основной целью ISPP является автоматизация каких-либо задач во время компиляции сценария и предотвращение возможных ошибок при его составлении. Например, можно объявить переменную ISPP (переменная, используемая в процессе компиляции) с именем приложения, а затем использовать её в различных местах сценария. Если по каким-то причинам нужно будет изменить имя приложения, то сделать это потребуется только один раз, изменив значение объявленной переменной. Во время компиляции сценария вместо этой переменной будет подставлено её значение. Без ISPP нужно было бы изменять все значения параметров, в которых используется имя приложения.

Другой пример использования ISPP – это сбор информации о версии приложения путем считывания данных из EXE-файла и подставновки их в директиву IS AppVerName секции [Setup] или где-то в другом месте сценария. Без применения ISPP в сценарии пришлось бы вручную менять все записи, в которых упоминается версия приложения.

Точно так же ISPP можно использовать для сканирования исходных папок и создания строк сценария со списком найденных файлов. Без ISPP все файлы установки придётся вносить вручную.

Кроме того, применяя директивы ISPP условной компиляции, имеется возможность исключить или наоборот выполнить часть сценария для различных версий/компоновок приложения (например, ознакомительная и полная версии). Без ISPP потребуется составить несколько различных сценариев.

И наконец, ISPP даёт возможность разбить длинные строки на более короткие, используя символ разделения строк.

Примечание: ISPP работает исключительно в время компиляции сценария и не имеет никакой функциональности при выполнении установки/деинсталляции приложения.
