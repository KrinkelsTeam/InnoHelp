Inno Setup Preprocessor: Insert

Прототип функции

void Insert(str *S, int Index, str Substr)


Описание

Выполняет вставку подстроки Substr, с позиции Index, в заданной строке S. Строковый параметр S передаётся в качестве ссылки, поэтому может быть изменён.

Функция объявлена в файле ISPPBuiltins.iss.
