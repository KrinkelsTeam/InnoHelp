Препроцессор Inno Setup: PackVersionNumbers

Прототип функции

int PackVersionNumbers(int VersionMS, int VersionLS)

Описание

Выполняет упаковку номеров версии в одно значение.

Всегда используйте ComparePackedVersion или SamePackedVersion для сравнения упакованных версий.

Функция объявлена в файле ISPPBuiltins.iss.
