Inno Setup Preprocessor: ReadIni

Прототип функции

str ReadIni(str 1, str 2, str 3, str? 4)


Описание

Выполняет чтение значений из INI-файла. Первый аргумент является именем INI-файла. Второй аргумент – это имя секции в заданном INI-файле. Третий аргумент – имя ключа, значение которого необходимо получить. Последний аргумент (4) является необязательным и может быть использован для установки значения, которое вернёт функция в случае неудачного выполнения операции или при пустом значении запрошенного ключа.
