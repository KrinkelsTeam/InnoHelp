Inno Setup Preprocessor: GetVersionNumbers

Прототип функции

str GetVersionNumbers(str Filename, int *VersionMS, int *VersionLS)

Описание

Получение номеров версии для заданного файла. В случае успеха возвращает строковое значение версии (в формате "0.0.0.0"), иначе — пустую строку.

Функция объявлена в файле ISPPBuiltins.iss.

PackVersionNumbers
