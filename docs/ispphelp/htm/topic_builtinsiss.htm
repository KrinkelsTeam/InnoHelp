Inno Setup Preprocessor: ISPPBuiltins.iss

Файл ISPPBuiltins.iss является неотъемлемой частью Inno Setup Preprocessor и устанавливается вместе с ним в корневой каталог Inno Setup. Он автоматически подключается к файлам сценариев во время предварительной обработки, т.к. содержит фактические реализации некоторых функций ISPP. Кроме того он содержит общие объявления переменных, такие как специальные константы. Практически файл ISPPBuiltins.iss представляет собой обычный сценарий Inno Setup, поэтому вы также можете использовать все его методы в собственных сценариях.
