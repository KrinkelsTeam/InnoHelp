Inno Setup Preprocessor: SetupSetting

Прототип функции

str SetupSetting(str)


Описание

Выполняет синтаксический анализ секции [Setup] в данных текущего вывода препроцессора с целью поиска ключа, имя которого задано в качестве аргумента функции (str). Функция возвращает значение этого ключа, если он найден, или пустую строку, если ключ не найден.
