Inno Setup Preprocessor: GetSHA1OfUnicodeString

Прототип функции

str GetSHA1OfString(str)


Описание

Возвращает строковое значение контрольной суммы SHA-1 для заданной строки в кодировке Unicode.

Внимание! Вызывает внутреннюю ошибку, если обращение к функции происходит в версии Inno Setup отличной от Unicode.
