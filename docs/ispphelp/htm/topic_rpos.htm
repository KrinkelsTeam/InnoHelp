Inno Setup Preprocessor: RPos

Прототип функции

int RPos(str 1, str 2)


Описание

Выполняет поиск подстроки (первый аргумент) в другой строке (второй аргумент) и возвращает целое значение, которое характеризует начальную позицию (по первому символу) искомой подстроки в пределах заданной строки. Если подстрока не найдена, то функция возвращает нулевое значение. Обратите внимание, что данная функция  чувствительна к регистру символов.
