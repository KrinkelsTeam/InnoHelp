Inno Setup Preprocessor: PackVersionComponents

Прототип функции

int PackVersionComponents(int Major, int Minor, int Revision, int Build)


Описание

Выполняет упаковку компонентов версии в одно значение.

Всегда используйте ComparePackedVersion или SamePackedVersion для сравнения упакованных версий.

Функция объявлена в файле ISPPBuiltins.iss.


См. также

GetVersionComponents, PackVersionNumbers, UnpackVersionComponents
