Inno Setup Preprocessor: Find

Прототип функции

int Find(int 1, str 2, int? 3, str? 4, int? 5, str? 6, int? 7)


Описание

Функция Find предназначена для использования совместно с директивой insert. Возвращает индекс строки в данных вывода препроцессора в зависимости от указанных критериев поиска.

Первый аргумент определяет индекс строки, с которой необходимо начать поиск. Обычно используется значение ноль (0).

Второй, четвертый и шестой аргументы определяют подстроку поиска в пределах каждой из строк. Второй аргумент является обязательным. Четвертый и шестой могут быть опущены.

Третий, пятый и седьмой аргументы определяют критерии поиска, которые задаются специальными флагами. Третий аргумент относится ко второму аргументу с подстрокой. Соответственно пятый параметр к четвертому и седьмой - к шестому.

Если любой из аргументов критерия поиска (флагов)  не указан, но определена соответствующая подстрока, то по умолчанию считается, что используются флаги FIND_MATCH | FIND_AND (т.е. равны 0).

Значения для третьего, пятого и седьмого аргументов функции Find объявлены в файле ISPPBuiltins.iss. См. раздел Флаги функции Find с описанием значений каждого флага.
