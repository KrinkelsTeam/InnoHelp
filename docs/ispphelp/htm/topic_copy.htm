Inno Setup Preprocessor: Copy

Прототип функции

str Copy(str 1, int 2, int? 3)


Описание

Выполняет извлечение подстроки из заданной строки, которая выступает в качестве первого аргумента. Второй аргумент указывает позицию первого символа, с которого начинается подстрока. Третий аргумент определяет количество необходимых символов в подстроке. Если он не указан, то в подстроку будут скопированы все символы от начальной позиции до конца строки.
