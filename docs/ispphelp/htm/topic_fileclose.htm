Inno Setup Preprocessor: FileClose

Прототип функции

void FileClose(int)

Описание

Выполняет закрытие файла, открытого с использованием функции FileOpen. После вызова FileClose дескриптор файла становится недействительным.

Примечание: если открытые файлы не смогут быть закрыты при помощи этой функции, то ISPP сделает это автоматически на этапе постобработки сценария.
