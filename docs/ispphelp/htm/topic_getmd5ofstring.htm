Inno Setup Preprocessor: GetMD5OfString

Прототип функции

str GetMD5OfString(str)


Описание

Возвращает строковое значение контрольной суммы MD5 для заданной строки в кодировке ANSI.


Пример

// Объявляем переменную MD5 и присваиваем ей значение
// контрольной суммы строки "Test", которое вернёт функция.
#define MD5 GetMD5OfString('Test')
// MD5 = '0cbc6611f5540bd0809a388dc95a615b'
