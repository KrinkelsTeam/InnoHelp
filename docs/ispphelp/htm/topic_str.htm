Inno Setup Preprocessor: Str

Прототип функции

str Str(any)


Описание

Выполняет преобразование заданного выражения в строку (строковый тип данных). Если выражение представляет собой целочисленное значение, то результатом функции будет его строковое представление. Если выражение является пустым (void), то функция вернёт пустую строку, в противном случае это будет значение выражения.
