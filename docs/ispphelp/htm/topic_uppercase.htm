Inno Setup Preprocessor: UpperCase

Прототип функции

str UpperCase(str)

Описание

Возвращает строку с тем же самым текстом, в котором все буквы преобразованы к верхнему регистру (прописные). Функция UpperCase выполняет преобразование только для символов в пределах "A" и "Z" из 7-бит таблицы ASCII.
