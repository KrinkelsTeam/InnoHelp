Inno Setup Preprocessor: GetVersionComponents

Прототип функции

str GetVersionComponents(str Filename, int *Major, int *Minor, int *Revision, int *Build)

Описание

Получение компонентов версии для заданного файла. В случае успеха возвращает строковое значение версии (в формате "0.0.0.0"), иначе — пустую строку.

Функция объявлена в файле ISPPBuiltins.iss.

PackVersionComponents
