Inno Setup Preprocessor: ComparePackedVersion

Прототип функции

int ComparePackedVersion(int Version1, int Version2)


Описание

Выполняет сравнение Version1 с Version2. Возвращает значение меньше 0, если Version1 меньше Version2; 0 - если Version1 равно Version2; больше 0 - если Version1 больше Version2.
