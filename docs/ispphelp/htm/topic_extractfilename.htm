Inno Setup Preprocessor: ExtractFileName

Прототип функции

str ExtractFileName(str PathName)


Описание

Выполняет извлечение имени файла из полного пути. Если PathName заканчивается обратной косой чертой, т.е. является путём к каталогу, то функция возвращает пустую строку.

Функция объявлена в файле ISPPBuiltins.iss.
