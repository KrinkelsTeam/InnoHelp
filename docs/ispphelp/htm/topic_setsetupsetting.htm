Inno Setup Preprocessor: SetSetupSetting

Прототип функции

void SetSetupSetting(str 1, str 2)


Описание

Выполняет создание или изменение в секции [Setup] директивы с заданным ключом (первый аргумент) и его значением (второй аргумент).

Если секция [Setup] в данных текущего вывода препроцессора отсутствует (в случае, если вызов этой функции был выполнен до объявления данной секции в сценарии), то такой заголовок (как и сама директива) будет создан автоматически.

Пожалуйста, при использовании функции SetSetupSetting будьте предельно осторожны. Не допускается её применение, когда директивы передаются ISPP встроенным методом (т.е. после использования директивы #insert).
