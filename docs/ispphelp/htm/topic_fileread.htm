Inno Setup Preprocessor: FileRead

Прототип функции

str FileRead(int)


Описание

Выполняет чтение строк файла, открытого с помощью функции FileOpen. Принимается только дескриптор файла, который возвращает функция FileOpen.


Пример

#define FileHandle
#define FileLine
#sub ProcessFileLine
  #define FileLine = FileRead(FileHandle)
  #pragma message FileLine
#endsub
#for {FileHandle = FileOpen("c:\autoexec.bat"); \
      FileHandle && !FileEof(FileHandle); ""} \
      ProcessFileLine
#if FileHandle
  #expr FileClose(FindHandle)
#endif

define, sub, pragma, for, if.
