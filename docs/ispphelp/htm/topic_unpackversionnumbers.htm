Inno Setup Preprocessor: UnpackVersionNumbers

Прототип функции

void UnpackVersionNumbers(int Version, int *VersionMS, int *VersionLS)

Описание

Выполняет распаковку упакованной версии в номера версии.

Функция объявлена в файле ISPPBuiltins.iss.
