Inno Setup Preprocessor: FindNext

Прототип функции

int FindNext(int)


Описание

Возвращает следующую запись, которая соответствует заданным критериям поиска, определённых при вызове функции FindFirst. Функция использует дескриптор поиска, который возвращает функция FindFirst. Если значение дескриптора не равно нулю, значит функция выполнена успешно.
